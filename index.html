プライバシーについて

fanbom はすべての処理をブラウザ内で完結します。録画データはサーバーへ送信されず、外部に一切共有されません。録画はブラウザのメモリ（RAM）にのみ保存され、タブを閉じると消去されます。

免責事項

本ツールの使用は自己責任で行ってください。開発者は録画内容・使用目的・第三者への影響に対して一切の責任を負いません。

録画に関する法律

第三者の映像・音声・画面を録画する行為は、国・地域によって法的規制の対象となる場合があります。本ツールを使用する前に、お住まいの地域の法令（録音・盗撮禁止法、個人情報保護法等）を確認し、必要な同意を取得した上でご利用ください。同意のない録画は法律に違反する可能性があります。

ブラウザ制限について

本ツールは Web API（getDisplayMedia / MediaRecorder）を使用しています。システム音声のキャプチャは Chrome のみ対応で、macOS では OS の制限により取得できない場合があります。録画ファイルは WebM 形式で保存されます（VLC、Chrome、Firefox で再生可能）。

fanbom

シンプルなブラウザ画面録画ツール

⚠

録画はブラウザのメモリに一時保存されます。タブを閉じる前に必ず「保存」してください。閉じると録画データは消えます。

ℹ

画面録画には HTTPS または localhost が必要です。このページは HTTP で開かれているため録画できない場合があります。

音声設定

🎙
マイク音声 マイクからの音声を録音する
🔊
システム音声 Chrome のみ対応 / 画面共有ダイアログで「音声を共有」にチェックが必要
🔇
音声なし 映像のみ録画する
システム音声について：
Chrome のみ対応。画面共有ダイアログで必ず「タブの音声を共有」または「音声を共有」にチェックしてください。macOS ではOS制限により取得できません。マイクと同時使用する場合は自動でミキシングされます。
